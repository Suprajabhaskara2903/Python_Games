How to play Pin ball?

Description: The user should direct the pin ball to fly in between two horizantal moving blocks. Failing this will lead to loose the game.
This game was built uisng python package - Turtle

How to play Snake Game?

Description: The user has to guide the snake to collect its food without letting the snake touch boundaries or walls. Consuming food will increase the snake's length.
This game was built using python package- Turtle, Time, Random
